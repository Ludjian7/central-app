Central Computers Management System

Welcome to the Central Computers Management System Demo. This application showcases a simplified version of a computer store management system.

This demo uses mock data and demonstrates various features typically found in retail management software.

Features

Sales Management

Track sales transactions, create invoices, and manage customer information.

Inventory Tracking

Monitor product stock levels, track prices, and manage product categories.

Financial Reporting

Generate reports on sales, revenue, expenses, and profit margins.

API Endpoints

This demo provides several API endpoints that return mock data:

API Base: /api

Overview of available API endpoints

Health Check: /api/health

API health status information

Sales Data: /api/sales

List of mock sales transactions

Products: /api/products

List of product inventory

Finance Reports: /api/finance/reports

Financial reporting data

Dashboard Summary: /api/dashboard/summary

Overview of key metrics
